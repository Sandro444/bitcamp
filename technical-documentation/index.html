Clockwise Model

Until we start defining our margins and paddings, it would be better, if we talked little bit about shorthand rules. Clockwise rule in css, allow us to define box model properties, for example margin, in one line. When defining property in one line, it is defined as follows: top, left, bottom, right. When defining to props, they will go like this: top&bottom, left&right.

margin: 5px 0 5px 0; padding: 5px 4px ;
Margin

Margin is outer space of the element. By defining margin, you can define the value, that will not be overriden by other element in normal flow. However, margin can be overriden by elements, that have position of absolute or fixed, but most of the time, it is not the case.

margin: 5px 0;
Border

Border is a part that is across the box. Border has properties: thickness, color, style, radius, etc. It is defined like this

border: 1px solid green;
border: 10px dotted red;
border: 3px groove blue;
Padding

Padding is space, that is between border and content. You can define padding with one line, or multiple lines.

padding: 1px 3px 5px 1px;
Content

Content is area, that is left over, after you define, or not define other properties. Content is main section of the box. It doesn't need defining

Border Box

There are two types of box sizing, border-box and content-box. Border box means, that your width includes borders, when defining box width.

box-sizing: border-box;

It is supported in all browsers

Content Box

There are two types of box sizing, border-box and content-box. Content box means, that your width doesn't include borders, when defining box width. It only includes content and padding

box-sizing: content-box;

It is supported in all browsers
